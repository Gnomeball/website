Swansea University Hack Society aims to provide a fun and collaborative space for students to work on projects. We run weekly meets on Wednesdays where members can come to work on projects that interest them. We provide support for students working on programming and technology related coursework. As well as offering intro to programming lessons and talks on exciting new areas of tech. We aim to run a major Hackathon event each semester with the goal of getting as many people in a room as possible working to make creative solutions to a set task.
